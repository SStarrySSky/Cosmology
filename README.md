# SRST Cosmology Code

Code for reproducing the numerical results of the paper:

**Qiucheng Gong**  
*Macroscopic Cosmological Dynamics from a Nested Delayed-Release Horizon Sector*

This repository contains the scripts used for the background fit, growth analysis, and profile-likelihood calculations described in the paper.

# Requirements

Python 3.14 

# Outputs

## The scripts generate:

summary JSON files

best-fit comparison plots

growth comparison plots

profile-likelihood plot for ε_g
